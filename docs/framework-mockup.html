✦ Why Astro Starlight

→ Dark-first design — matches the workflow guide aesthetic you already loved
→ Full CSS/component control — can match any visual style
→ Better landing page & hero support out of the box
→ Faster build times, smaller output
→ Vercel deploys in one click

◈ Why Docusaurus

→ More familiar to developers from open-source docs
→ Built-in versioning (useful if guide evolves rapidly)
→ React-based — easier if the team knows React
→ Algolia search integration is mature
→ "Looks like docs" — professional, no-surprise layout

Verdict for this project

→ Astro Starlight — the guide is visual-first and design matters to the audience
→ Can replicate the dark card aesthetic from the HTML guide exactly
→ Beginner audience will find a beautiful, modern site more inviting
→ One command to scaffold: npm create astro@latest -- --template starlight
